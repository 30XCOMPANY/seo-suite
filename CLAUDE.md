# 30X SEO Suite

AI-powered SEO automation for Claude Code. 50+ skills across 3 suites.

## Key Commands
- `/seo audit <url>` — Full site audit
- `/seo geo <url>` — GEO/AI visibility
- `/seo technical <url>` — Technical SEO
- `/seo content <url>` — Content + E-E-A-T
- `/seo brief <url>` — Content brief generator
- `/seo plan <url>` — Competitor analysis + roadmap

## Structure
- `SKILL.md` — Master orchestrator
- `skills/` — Sub-skills directory
- `index.html` — Landing page (deploys via GitHub Actions)
